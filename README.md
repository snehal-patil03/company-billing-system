🧾 Company Billing System

📌 Overview
A GST-based billing system developed to automate billing operations and manage customer, product, and transaction data efficiently.

🚀 Features
- Customer Management
- Product Management
- GST Invoice Generation (CGST, SGST, IGST)
- Payment Tracking
- Reports Generation

 🛠 Technologies
HTML, CSS, Bootstrap, JavaScript, PHP, MySQL

 🎯 Objective
To reduce manual work and improve billing accuracy.
